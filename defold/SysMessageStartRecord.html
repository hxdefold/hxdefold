typedef SysMessageStartRecord

package defold

import defold.Sys

Data for the SysMessages.start_record message.

@:optionaloptionalframe_period:Null<Int>

Frame period to record, ie write every nth frame. Default value is 2.

@:optionaloptionalfps:Null<Int>

Frames per second. Playback speed for the video. Default value is 30.

The fps value doens't affect the recording. It's only meta-data in the written video file.

file_name:String

File name to write the video to.
